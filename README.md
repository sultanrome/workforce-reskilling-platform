# Workforce Reskilling Platform

A role-based platform built in Java to support people affected 
by AI and automation by helping them learn new skills and 
connect with job opportunities.

## Tech Stack
- Java
- JavaFX
- Spring Boot
- Maven
- SQL Database
- Object Oriented Programming

## User Roles
- Workers — register, enrol in training programs, track 
  learning progress and search for jobs
- Trainers — manage courses, monitor worker performance 
  and track training effectiveness
- Employers — post jobs, review skilled workers and 
  monitor recruitment progress

## Features
- Login and registration system
- Role-based dashboards using JavaFX GUI
- Training program management
- Job posting and application system
- SQL database design
- UML and ER diagrams

## About
Built as part of BSc Computer Science at University of 
Bedfordshire. Demonstrates Java OOP, GUI development with 
JavaFX and database integration.
